# growi-plugin-bookshelf-map

GROWI のページツリーを「本棚ビュー」で表示するスクリプトプラグインです。サイドバーに「📚 本棚」ボタンが追加され、クリックでパネルが開き、指定パス配下のページをカード一覧で表示します。

## 機能（Phase 1）

- **本棚ビュー**: 現在ページのパス（またはルート）を「棚」として、その直下の子ページをカード表示
- **戻る / ルート**: パネル内で棚を掘り下げたあと、戻る・ルートで表示を切り替え
- **カードクリック**: ページへ遷移（子を持つページはクリックでその棚を開く予定）

## 必要な環境

- GROWI（オンプレ / GROWI.cloud）
- ブラウザから `/_api/v3/pages/list` が同一オリジンで利用できること

## インストール

1. このリポジトリを GROWI のプラグインとしてインストール（管理画面の「プラグイン」でリポジトリ URL を指定）
2. プラグインを有効化
3. サイドバーに「📚 本棚」が表示されるのでクリックしてパネルを開く

## 開発

```bash
npm install
npm run build
```
